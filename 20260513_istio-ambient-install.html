Istio Ambient v1.29.0 安装配置

模式: Ambient（无 sidecar）/ 镜像: Harbor devops81

0. 前提条件（两个节点务必先执行）

mkdir -p /opt/cni/bin && ln -sf /var/lib/rancher/k3s/data/current/bin/istio-cni /opt/cni/bin/istio-cni

1. 下载 istioctl

curl -L https://istio.io/downloadIstio | ISTIO_VERSION=1.29.0 sh -
echo 'export PATH=$PATH:/root/istio-1.29.0/bin' >> /etc/profile
source /etc/profile

2. 创建命名空间

kubectl create namespace istio-system

3. istio-ambient.yaml

apiVersion: install.istio.io/v1alpha1
kind: IstioOperator
metadata:
  namespace: istio-system
spec:
  hub: harbor.devops.local:7581/devops81
  tag: 1.29.0
  profile: ambient
  components:
    ingressGateways:
    - name: istio-ingressgateway
      enabled: true
    pilot:
      k8s:
        resources:
          requests:
            memory: 200Mi
            cpu: 100m
  values:
    global:
      proxy:
        resources:
          requests:
            memory: 64Mi
            cpu: 10m
    cni:
      cniBinDir: /var/lib/rancher/k3s/data/current/bin
      cniConfDir: /etc/cni/net.d
      excludeNamespaces:
        - kube-system
        - istio-system
⚠️ 关键：excludeNamespaces 必须加 [kube-system, istio-system]，否则 istio 自身组件创建 sandbox 会卡死

4. 安装

istioctl install -f ~/istio-ambient.yaml -y

验证

kubectl get pods -n istio-system
# 预期 6/6 Running: istiod, istio-cni-node(x2), ztunnel(x2), istio-ingressgateway
