istioctl 安装

source /etc/profile

安装步骤

1. istio-base

kubectl create namespace istio-system

helm repo add istio https://istio-release.storage.googleapis.com/charts
helm repo update

helm install istio-base istio/base \
  --version 1.24.3 \
  --namespace istio-system

2. istiod

helm install istiod istio/istiod \
  --version 1.24.3 \
  --namespace istio-system \
  --set pilot.image=harbor.devops.local:7581/devops111/istio-pilot:1.24.3 \
  --set pilot.imagePullPolicy=IfNotPresent \
  --set global.proxy.image=harbor.devops.local:7581/devops111/istio-proxyv2:1.24.3 \
  --set global.proxy.imagePullPolicy=IfNotPresent \
  --set profile=ambient \
  --set global.imagePullPolicy=IfNotPresent

3. istio-cni（用 istioctl）

istioctl install --set profile=ambient \
  --set components.cni.enabled=true \
  --set values.cni.image=harbor.devops.local:7581/devops111/istio-install-cni:1.24.3 \
  --set values.cni.cniBinDir=/var/lib/rancher/k3s/data/current/bin \
  --set values.cni.cniConfDir=/etc/cni/net.d \
  --set values.cni.chained=true \
  --set values.pilot.image=harbor.devops.local:7581/devops111/istio-pilot:1.24.3 \
  --set values.global.proxy.image=harbor.devops.local:7581/devops111/istio-proxyv2:1.24.3 \
  -y

4. ztunnel

helm install ztunnel istio/ztunnel \
  --version 1.24.3 \
  --namespace istio-system \
  --set image=harbor.devops.local:7581/devops111/istio-ztunnel:1.24.3 \
  --set imagePullPolicy=IfNotPresent

关键参数说明

参数说明
profile=ambient启用 Ambient 模式，无 sidecar
cni.chained=true链式 CNI，不抢占 Cilium 主 CNI
cni.cniBinDirK3s CNI bin 路径 /var/lib/rancher/k3s/data/current/bin
ztunnel每节点 DaemonSet，负责 L4 透明加密和流量转发

与 Cilium 共存关键点

  1. Cilium 必须设置 cni.exclusive=false 并重启 DaemonSet
  2. istio-cni 使用 chained=true，插入 Cilium conflist 而不是替换
  3. K3s CNI bin 路径非标准，必须指定
  4. 安装顺序：istiod → istio-cni → ztunnel
