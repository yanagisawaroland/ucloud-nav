K3s 安装配置说明

版本: K3s v1.31.5+k3s1 / Kubernetes 1.31

节点信息

角色主机名IP
Servermaster11110.20.50.111
Agentnode11210.20.50.112

master111 安装命令

export http_proxy=http://10.20.50.4:10809
export https_proxy=http://10.20.50.4:10809
curl -sfL https://get.k3s.io | INSTALL_K3S_VERSION="v1.31.5+k3s1" \
  sh -s - server \
  --disable traefik \
  --disable servicelb \
  --disable-kube-proxy \
  --flannel-backend=none \
  --cluster-cidr=10.42.0.0/16 \
  --service-cidr=10.43.0.0/16 \
  --node-ip=10.20.50.111 \
  --advertise-address=10.20.50.111 \
  --write-kubeconfig-mode=644
unset http_proxy https_proxy

node112 安装命令

export http_proxy=http://10.20.50.4:10809
export https_proxy=http://10.20.50.4:10809
curl -sfL https://get.k3s.io | INSTALL_K3S_VERSION="v1.31.5+k3s1" \
  sh -s - agent \
  --server https://10.20.50.111:6443 \
  --token <node-token> \
  --node-ip=10.20.50.112
unset http_proxy https_proxy

关键参数说明

参数说明
--disable traefik禁用默认 Ingress，由 Istio 接管
--disable servicelb禁用默认 LB
--disable-kube-proxy禁用 kube-proxy，由 Cilium eBPF 替代
--flannel-backend=none禁用 flannel，由 Cilium 接管 CNI
--cluster-cidrPod 网段 10.42.0.0/16
--service-cidrService 网段 10.43.0.0/16

Harbor 认证配置（两个节点都需要）

文件路径：/etc/rancher/k3s/registries.yaml

mirrors:
  harbor.devops.local:7581:
    endpoint:
      - "http://harbor.devops.local:7581"
configs:
  "harbor.devops.local:7581":
    auth:
      username: admin
      password: Harbor12345
    tls:
      insecure_skip_verify: true

修改后重启：

# master111
systemctl restart k3s
# node112
systemctl restart k3s-agent

环境变量

echo 'export KUBECONFIG=/etc/rancher/k3s/k3s.yaml' >> /etc/profile
source /etc/profile
