Cilium 安装配置说明

版本: Cilium v1.16.6

Harbor 镜像准备（node75）

export http_proxy=http://10.20.50.4:10809
export https_proxy=http://10.20.50.4:10809
docker pull m.daocloud.io/quay.io/cilium/cilium:v1.16.6
docker pull m.daocloud.io/quay.io/cilium/operator-generic:v1.16.6
unset http_proxy https_proxy

docker tag m.daocloud.io/quay.io/cilium/cilium:v1.16.6 harbor.devops.local:7581/devops111/cilium-cilium:v1.16.6
docker tag m.daocloud.io/quay.io/cilium/operator-generic:v1.16.6 harbor.devops.local:7581/devops111/cilium-operator-generic:v1.16.6
docker push harbor.devops.local:7581/devops111/cilium-cilium:v1.16.6
docker push harbor.devops.local:7581/devops111/cilium-operator-generic:v1.16.6

Helm 安装

helm repo add cilium https://helm.cilium.io/
helm repo update

helm install cilium cilium/cilium \
  --version 1.16.6 \
  --namespace kube-system \
  --set image.repository=harbor.devops.local:7581/devops111/cilium-cilium \
  --set image.tag=v1.16.6 \
  --set image.pullPolicy=IfNotPresent \
  --set image.useDigest=false \
  --set operator.image.repository=harbor.devops.local:7581/devops111/cilium-operator \
  --set operator.image.tag=v1.16.6 \
  --set operator.image.pullPolicy=IfNotPresent \
  --set operator.image.useDigest=false \
  --set operator.image.suffix="" \
  --set kubeProxyReplacement=true \
  --set k8sServiceHost=10.20.50.111 \
  --set k8sServicePort=6443 \
  --set ipam.mode=kubernetes \
  --set cni.exclusive=false

关键参数说明

参数说明
kubeProxyReplacement=true用 eBPF 完全替代 kube-proxy
k8sServiceHost/PortAPIServer 地址，kubeProxyReplacement 必填
ipam.mode=kubernetes使用 K8s 原生 IPAM
cni.exclusive=false允许 Istio CNI 链式插入，与 Istio Ambient 共存必须设置
image.useDigest=false禁用 digest 校验，使用 Harbor 私有镜像必须设置
operator.image.suffix=""清空 operator 镜像后缀，防止自动拼接 -generic

与 Istio Ambient 共存注意事项

cni.exclusive=false 必须设置，否则 Cilium 阻止 istio-cni 写入 conflist
kubectl rollout restart daemonset/cilium -n kube-system
