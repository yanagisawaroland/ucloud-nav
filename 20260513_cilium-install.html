Cilium v1.19.1 安装配置

模式: kubeProxyReplacement=true / cni.exclusive=false

安装命令

export KUBECONFIG=/etc/rancher/k3s/k3s.yaml

helm repo add cilium https://helm.cilium.io/
helm repo update

helm install cilium cilium/cilium \
  --version 1.19.1 \
  --namespace kube-system \
  --set image.override=harbor.devops.local:7581/devops81/cilium-cilium:v1.19.1 \
  --set image.pullPolicy=IfNotPresent \
  --set image.useDigest=false \
  --set operator.image.override=harbor.devops.local:7581/devops81/cilium-operator-generic:v1.19.1 \
  --set operator.image.pullPolicy=IfNotPresent \
  --set operator.image.useDigest=false \
  --set kubeProxyReplacement=true \
  --set k8sServiceHost=10.20.50.81 \
  --set k8sServicePort=6443 \
  --set ipam.mode=kubernetes \
  --set cni.exclusive=false \
  --set envoy.image.override=harbor.devops.local:7581/devops81/cilium-envoy:v1.35.9-1770979049-232ed4a26881e4ab4f766f251f258ed424fff663

关键参数说明

参数说明
kubeProxyReplacement=trueeBPF 替代 kube-proxy，配合 --disable-kube-proxy
k8sServiceHost/PortAPIServer 地址，kubeProxyReplacement 必填
cni.exclusive=false允许 Istio CNI 链式插入
image.useDigest=falseHarbor 私有镜像必须设置
registries.yaml 必须提前配好，用 http:// 协议，重启后生效
